## 🚀 Features

- 🎥 Detects deepfake videos using AI models  
- 🧠 CNN + ResNet18 based architecture  
- ⚡ Real-time prediction via web interface  
- 📊 Confidence score for predictions  
- 🔐 Helps prevent misinformation and fraud

- Python

- Frames are extracted and processed  
- CNN/ResNet model analyzes features  
- Outputs prediction (Real / Fake)  
- Displays confidence score
